Image Field Demo

단일/다중 이미지 업로드 필드 컴포넌트입니다. multiple, disabled, allowDrop, readMode, maxSizeBytes 등의 속성을 확인할 수 있습니다.

Basic

가장 기본적인 이미지 업로드 예시입니다. 업로드 후 각 항목의 미리보기 버튼으로 이미지 뷰어 모달을 열 수 있습니다.

단일 이미지 업로드용 기본 예시입니다.

State & Control

disabled 상태와 외부 제어 예시입니다.

disabled 속성으로 선택/드롭 동작을 막을 수 있습니다.

Multiple

여러 이미지를 업로드하고 최대 개수 제한을 적용하는 예시입니다.

최대 3개까지 업로드할 수 있습니다.

Read Mode

readMode="dataUrl" 와 readMode="objectUrl" 차이를 확인합니다.

dataUrl은 문자열 데이터로 읽어 바로 바인딩할 때 사용합니다.
objectUrl은 브라우저 URL을 만들어 미리보기할 때 유리합니다.

Drag & Drop

allowDrop 속성으로 드래그 앤 드롭 허용 여부를 제어합니다.

드롭 가능 상태입니다.
드롭 비허용 상태입니다.

File Rules

파일 형식과 최대 용량 제한 예시입니다.

accept 속성으로 허용 파일 형식을 제한합니다.
maxSizeBytes 속성으로 최대 용량을 제한합니다.
